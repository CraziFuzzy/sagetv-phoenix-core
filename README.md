# sagetv-phoenix-core

Phoenix Core Services for SageTV

## Building Phoenix From Source
* Checkout the code
* Run `./gradlew configure`

If using Eclipse
* Run `./gradlew cleanEclipse eclipse`
* Import project

If using IntelliJ
* Open `build.gradle` as project file
